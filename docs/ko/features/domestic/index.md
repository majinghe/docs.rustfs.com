# 국산 클라우드 솔루션

RustFS는 국산 클라우드 환경을 위한 포괄적인 솔루션을 제공하며, 주요 중국 클라우드 제공업체를 지원하고 데이터 주권에 대한 규정 준수 요구사항을 충족합니다.

## 주요 기능

### 데이터 보안

- **저장 시 암호화**: 저장된 데이터에 대한 AES-256 암호화
- **전송 중 암호화**: 모든 데이터 전송에 대한 TLS 1.3
- **키 관리**: 국산 KMS 솔루션과 통합
- **액세스 제어**: 세밀한 권한 및 감사 로깅

### 성능 최적화

- **네트워크 최적화**: 중국 인터넷 인프라에 최적화
- **엣지 컴퓨팅**: 엣지 배포 지원
- **캐싱**: 향상된 성능을 위한 지능형 캐싱
- **로드 밸런싱**: 자동 부하 분산

### 규정 준수 및 거버넌스

- **데이터 분류**: 자동 데이터 분류 및 라벨링
- **보존 정책**: 구성 가능한 데이터 보존 및 삭제
- **감사 로깅**: 포괄적인 감사 추적
- **보고**: 자동화된 규정 준수 보고
