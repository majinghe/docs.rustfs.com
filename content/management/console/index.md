The **RustFS Console** is the web interface for routine administration. Use this section to open the Console and find the management workflows available in the documentation.

## Open the Console

The Console listens on port `9001` by default. Open the following address in a browser, replacing `<server-ip>` with the RustFS server address:

```text
http://<server-ip>:9001
```

Sign in with the credentials configured for your RustFS deployment. For a guided first session, see [Console: First Steps](/installation/console-first-steps).
